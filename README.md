# skillswap-olivia
Project Title: Skill Swap
Project Description: Skill Swap is a website/application that connects skill seekers (students) with  skill providers (tutors) by providing a platform that allows for skills to be offered and utilized.
Technologies: This project will use HTML, CSS, and JavaScript for the frontend layer; Node.js and API endpoints for the backend; and, Sequelize ORM and data models for the database layer.
